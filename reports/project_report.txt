        AI-BASED FAKE NEWS DETECTION SYSTEM
              PROJECT REPORT

================================================================
1. ABSTRACT
================================================================

This project presents an AI-Based Fake News Detection System that
automatically classifies news articles as real or fake using
Machine Learning and Natural Language Processing techniques.

The system uses Logistic Regression trained on 44,898 news
articles from the Kaggle Dataset, achieving 99.14% accuracy.

A modern desktop GUI built with CustomTkinter provides real-time
predictions, confidence scores, prediction history, and an
analytics dashboard with visual charts.

================================================================
2. PROBLEM STATEMENT
================================================================

Fake news spreads rapidly through social media and digital
platforms, causing serious harm to society. Manual fact-checking
is slow and impossible to scale.

This project solves that problem by building an automated
AI system that instantly analyzes news content and determines
its authenticity with 99.14% accuracy.

4. DATASET
================================================================

Source         : Kaggle
Dataset Name   : Fake and Real News Dataset
Fake Articles  : 23,481  (Label = 0)
Real Articles  : 21,417  (Label = 1)
Total Articles : 44,898
Features Used  : Title + Text (combined)

================================================================
5. NLP PREPROCESSING PIPELINE
================================================================

Step 1 : Lowercase conversion
         "The GOVERNMENT" -> "the government"

Step 2 : URL Removal
         "Visit https://fake.com" -> "Visit"

Step 3 : Special Character Removal
         "fraud!!!" -> "fraud"

Step 4 : Stopword Removal
         "the government is lying" -> "government lying"

Step 5 : Porter Stemming
         "running" -> "run"
         "elections" -> "elect"

Result : 44,898 articles -> 44,889 clean articles
         (9 empty articles removed)

================================================================
6. FEATURE ENGINEERING - TF-IDF
================================================================

Method         : TF-IDF Vectorization
Vocabulary     : 50,000 features
N-gram Range   : (1, 2) - unigrams and bigrams
Training Shape : (35,911 x 50,000)

TF  = Word count in document / Total words in document
IDF = log(Total documents / Documents containing word)
TF-IDF = TF x IDF

Common words get LOW score  (not important)
Rare words get HIGH score   (very important)

================================================================
7. MODEL TRAINING
================================================================

Train-Test Split : 80% Training / 20% Testing
Training Samples : 35,911
Testing Samples  : 8,978

--- MODEL 1: LOGISTIC REGRESSION ---
Algorithm    : Logistic Regression
C            : 1.0 (regularization)
max_iter     : 1000
solver       : lbfgs

--- MODEL 2: NAIVE BAYES ---
Algorithm    : Multinomial Naive Bayes
alpha        : 0.1 (smoothing)
